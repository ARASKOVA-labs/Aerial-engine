# Security Policy

## Supported Versions

The following versions of Aerial Engine are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Aerial Engine, please send an e-mail to security@araskova-labs.com. All security vulnerabilities will be promptly addressed.

Please do **not** report security vulnerabilities via public GitHub issues.
